// Anjo payroll systen

client - React app
Server - node.js + Express
Database - Mongodb atlas

Front end setup

1. Login Page
	•	Simple form with validation
	•	API integration to /auth/login
	•	Store JWT token in localStorage

2. Sidebar & Routing
	•	Use react-router-dom for page routing
	•	Persistent sidebar with active state highlight

3. Dashboard
	•	Fetch monthly payroll data (bar chart)
	•	Display total spending and net average

4. Employees Page
	•	Form to add employee
	•	List view to show employee data
	•	POST & GET APIs integrated

5. Salary Calculation Page
	•	Dropdowns for employee and month
	•	Inputs for earnings/deductions
	•	Calculate gross, deductions, and net pay
	•	Save calculation to DB via POST

6. Reports Page
	•	Filter by month
	•	Show employee salary data
	•	Button to download payslip (PDF)

Backend setup

1. User Auth (Admin only)
	•	/auth/login → validate credentials → return JWT
	•	Middleware: authMiddleware.js to verify JWT

2. Employee Routes
	•	POST /employees → Add new
	•	GET /employees → List all

3. Salary Routes
	•	POST /salary/calculate → Save earnings, deductions, total
	•	GET /salary/:month → Return salaries for report

4. Reports/PDF
	•	Use html-pdf, pdfkit, or puppeteer to generate payslip
	•	Provide Download link for payslip
